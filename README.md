# Customer Segmentation using RFM Analysis

## Project Overview

This project applies RFM Analysis to segment customers from the Olist Brazilian E-Commerce dataset based on purchasing behavior.

RFM stands for:

- Recency: How recently a customer purchased
- Frequency: How often a customer purchased
- Monetary: How much a customer spent

The goal of this project is to identify customer groups such as Loyal Customers, Regular Customers, New Customers, At Risk Customers, and Churn Risk Customers, then provide targeted marketing recommendations for each segment.

## Tools Used

- Power Query
- Python
- Google Colab
- Power BI

## Project Workflow

1. Cleaned and prepared transactional customer data using Power Query.
2. Merged customer, order, and payment data.
3. Calculated RFM metrics using Python.
4. Created customer segments based on RFM scores.
5. Built a Power BI dashboard to visualize customer segments.
6. Provided targeted marketing recommendations.

## Customer Segments

- Loyal Customers
- Regular Customers
- New Customers
- At Risk Customers
- Churn Risk Customers

## Key Insights

- Regular Customers contributed the highest revenue.
- Loyal Customers had the highest average spending and purchase frequency.
- Churn Risk Customers had the longest period since last purchase.
- At Risk and Churn Risk Customers should be prioritized for retention campaigns.
- New Customers present an opportunity for growth through onboarding campaigns.

## Marketing Recommendations

- Loyal Customers: Reward with loyalty programs and exclusive offers.
- Regular Customers: Encourage repeat purchases with personalized bundles.
- New Customers: Use welcome campaigns to encourage second purchases.
- At Risk Customers: Offer reminders and limited-time discounts.
- Churn Risk Customers: Run win-back campaigns with strong incentives.
